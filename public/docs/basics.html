Alert

Button styles can be applied to any element. Typically you'll want to use either a <button> or an <a> element:

  • Code
  • Example
Example alert 
<div class="alert-container">
  <div class="alert">
    <div class="inner">
      <span>Example alert</span>
      <button class="js-alert-close">x</button>
    </div>
  </div>
</div>

Switch

Basic switch styles ideally used with just 2 links, but it will work with more if needed. To align in the middle use the .middle class and to highlight the selected link use the .selected class on the <a> tag.

  • Code
  • Example
  • Placepot
  • League
<ul class="switch">
  <li><a href="" class="selected">Placepot</a></li>
  <li><a href="">League</a></li>
</ul>
